Gradients

Linear-Gradients

Linear Gradients are the most basic type of gradients. Gradients work as background images, and a linear gradient forms colors in a defined order from top to bottom. The colors fade in and out, making it seem like a rainbow.

Radial Gradients

Radial Gradients are similar to Linear Gradients, except the colors begin at a single point and expand outwards. The point could be at any place in the gradient: the center, top-left, etc. The default shape of the color gradient is an ellipse, unless the image is square. Look Below for more examples.
